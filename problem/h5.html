# 字体大小不受手机的影响

有的人手机字体设置比较大，这种用户打开的 H5 项目 布局会乱掉

  • 此方法 适用于微信里打开的 H5 项目

  • iso 手机端的处理

html,
body .app {
	color: #333333;
	font-family: Arial, Helvetica, 'STHeiti STXihei', 'Microsoft YaHei', Tohoma, sans-serif;
	background-color: $background-color;
	/* ios 限制微信更改字体大小 */
	-webkit-text-size-adjust: 100% !important;
}
  • 安卓手机的处理

此文件在 man.js 引入即可

  • 可以将这个文件放在 utils 下

utils/font-rest.js

// 解决安卓端字体大小 不跟随系统变化的问题
// 如果 eslint 报错 说 WeixinJSBridgeReady 不存在
// globals: { wx: true,WeixinJSBridge:true },
;(function () {
	if (typeof WeixinJSBridge === 'object' && typeof WeixinJSBridge.invoke === 'function') {
		handleFontSize()
	} else {
		document.addEventListener('WeixinJSBridgeReady', handleFontSize, false)
	}
	function handleFontSize() {
		// 设置网页字体为默认大小
		WeixinJSBridge.invoke('setFontSizeCallback', {fontSize: 0})
		// 重写设置网页字体大小的事件
		WeixinJSBridge.on('menu:setfont', function () {
			WeixinJSBridge.invoke('setFontSizeCallback', {fontSize: 0})
		})
	}
})()

# 隐藏滚动条

不管是横向滚动或者 纵向滚动 手机端上的滚动条怎么看都不好看

conent {
	overflow-x: auto;
	// 横向滚动必须设置容器的宽度
	width:200px； &::-webkit-scrollbar {
		// 隐藏滚动条 重点
		display: none;
	}
}

# 滚动穿透的处理

  • 这个是 vue 版本的 原理就是通过 监听 弹出框是否正在使用来控制 body 中的内容不可以滚动
  • 一定要记得 弹出框关闭 要将 body 的 overflow 复原 不然 页面就动不了
export default {
	data() {
		return {
			overlayShow: false,
		}
	},
	watch: {
		overlayShow: {
			handler(newValue) {
				if (newValue) {
					document.body.style.overflow = 'hidden'
				} else {
					document.body.style.overflow = 'auto'
				}
			},
		},
	},
}

# h5 下载操作

iframe 是 手机端下载 兼容性最好的处理方案

function downloadH5(row) {
	let iframe = document.createElement('iframe')
	iframe.style.display = 'none'
	iframe.src = 'javascript: \'<script>location.href="' + row + '"</script>\''
	document.getElementsByTagName('body')[0].appendChild(iframe)
}

downloadH5('需要下载的内容')
